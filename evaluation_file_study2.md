## [섹션 4] 가상 환자(VP)의 시각/언어적 자연스러움 평가

( ) 1점 - 전혀 사람 같지 않음 (로봇처럼 기계적이고 각인된 톤, 잦은 부자연스러움)
( ) 2점 - 다소 기계적임
( ) 3점 - 보통 (사람 같지만 때때로 정해진 각본이나 AI처럼 느껴짐)
( ) 4점 - 대체로 사람 같음
( ) 5점 - 매우 사람 같음 (실제 인간 상호작용에서 볼 수 있는 미묘한 뉘앙스와 톤 변화를 보임)

**18. [시각적 자연스러움] 가상 환자의 표정, 시선 처리, 입모양, 신체 움직임이 얼마나 자연스러웠습니까?**
( ) 1점 - 매우 부자연스러움 (표정이나 움직임이 맥락과 맞지 않고 이질감이 큼)
( ) 2점 - 다소 부자연스러움
( ) 3점 - 보통
( ) 4점 - 대체로 자연스러움
( ) 5점 - 매우 자연스러움 (실제 사람의 비언어적 행동과 완벽하게 일치함)

**19. [발화의 유창성] 가상 환자의 말이 끊기지 않고 맥락에 맞게 유창하게 이어졌습니까?**
( ) 1점 - 전혀 유창하지 않음 (의미 없는 단절, 불완전한 문장 반복)
( ) 2점 - 다소 유창하지 않음
( ) 3점 - 보통
( ) 4점 - 대체로 유창함
